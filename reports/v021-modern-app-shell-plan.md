# Diet Pro Planner v0.0.21 — Modern App Shell & Design System

## Goal

Make Diet Pro Planner feel like a modern personal health app instead of a legacy administrative dashboard, without changing backend behavior or private data.

## Phase 1 scope

- Modernize the application shell.
- Reduce the visual weight of the desktop sidebar.
- Introduce a compact mobile bottom navigation.
- Standardize typography, spacing, radii, shadows, buttons and form fields.
- Improve page headers and content width.
- Preserve all existing routes, data and workflows.
- Avoid growth in `static/app.js` and `static/styles.css`.

## Non-goals

- No backend changes.
- No database migrations.
- No new AI provider integration.
- No dashboard information-architecture rewrite yet.
- No new nutrition or sport features.

## Follow-up phases

1. Dashboard Today 2.0.
2. Smart Coach visual cards and action-first flow.
3. Fast meal logging.
4. Weight and activity progress visual redesign.
